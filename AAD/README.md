# Opage

> Homepage for your browser

`Opage` 是一个客制化上网首页

基于 Material Design & Vue.js

集成搜索，网站，天气，TODO

Faster & Pure

## 功能   

- 搜索：Google + Bing + 百度
- 网站：可自定义网站列表
- 天气：全国城市 + 4 天预测
- TODO: 简单的 Todo 列表

## 使用

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
